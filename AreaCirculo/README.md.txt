# Área do Círculo

Este projeto consiste em um programa Java que calcula a área de um círculo a partir do raio informado pelo usuário.

O programa:
- Solicita ao usuário que digite o raio.
- Calcula a área usando a fórmula A = pi * r².
- Exibe o resultado formatado com 4 casas decimais.

É um exercício inicial para praticar:
- Entrada de dados com Scanner
- Operações matemáticas básicas
- Formatação de saída
- Boas práticas de código
